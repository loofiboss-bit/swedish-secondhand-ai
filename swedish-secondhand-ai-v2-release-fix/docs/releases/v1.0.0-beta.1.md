# Swedish Secondhand AI v1.0.0-beta.1

This is the first public desktop beta for the direct v1 release train.

## Highlights

- Safe-by-default first-run onboarding with deterministic offline analysis, optional local Ollama,
  or explicitly selected Gemini cloud analysis.
- OS-protected Gemini and Tradera secrets behind narrow validated Electron IPC; secrets are never
  included in renderer state, diagnostics, or backups.
- User-correctable, source-labelled product facts and evidence-gated valuation. Insufficient
  evidence produces no invented price.
- Versioned local datasets with idempotent v0.5 migration, validated secret-free backup/import,
  and selective/full recovery.
- Verified Tradera, Blocket, and Vinted templates that preserve defects, missing accessories,
  untested status, and authenticity uncertainty.

## Downloads

- Linux x64 AppImage.
- Windows x64 NSIS installer and portable executable.
- CycloneDX JSON SBOM and `SHA256SUMS.txt` for every downloadable artifact.

Windows packages are intentionally unsigned because this project has no code-signing certificate.
Windows may therefore show a SmartScreen warning. Verify the matching SHA-256 checksum before use.

## Beta expectations

Back up local data before upgrading. v0.5 data is migrated in place and migration is idempotent,
but this prerelease should still be treated as beta software. Report reproducible defects using
the process in `SUPPORT.md`; security issues should follow `SECURITY.md`.

Stable `v1.0.0` will not be published until this beta has been public for at least seven calendar
days and has no open P0/P1, high-severity security, or data-loss findings.
