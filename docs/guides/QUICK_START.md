# Quick start

You can create a listing locally without an account, API key or market data. The app never
publishes for you; review and copy the result to Tradera, Blocket or Vinted.

## Install

Download the Windows or Linux package from
[GitHub Releases](https://github.com/loofiboss-bit/swedish-secondhand-ai/releases). Verify its
SHA-256 checksum before opening it. Windows packages are unsigned and may trigger SmartScreen.

Read [Install and update](./INSTALLATION.md) for exact commands.

## Create your first listing

1. Select **Start offline**.
2. Select **New item**, enter a project name and factual description, then select
   **Create and continue**.
3. Select **Identify item**. Correct errors and mark uncertain information as unknown.
4. Open **Price**, enter **Your price**, then select **Use my price**.
5. Open **Listing** and select **Update untouched fields**.
6. Review the marketplace tab and copy the structured package.

Publish manually on the marketplace. Always verify condition, defects, accessories, test status
and authenticity before pasting the text.

## Choose a price path

**Your price** is the shortest path. It is clearly labelled as yours and receives no invented
confidence or market claim.

An evidence-based recommendation requires at least two realized prices that you approve. Active
asking prices show seller expectations only and never affect the numeric recommendation.

## Back up and recover

Export a full backup before upgrading or restoring data. Backups never include API keys. Deleted
projects first move to recoverable trash. **Empty trash** permanently deletes its contents after
an additional confirmation.

## Get help

Start with [Troubleshooting](./TROUBLESHOOTING.md). Attach only the app's safe diagnostic file and
sanitized reproduction steps to a public GitHub issue. Never post API keys, private listing URLs,
images or unpublished descriptions.
