# Swedish Secondhand AI v3.0.0

This stable release delivers **Guided Selling** for a copy-ready local listing without an account,
API key or market data.

## Guided offline path

- Start offline and work through **Item → Price → Listing → Done**.
- Choose a seller-entered price, an optional evidence-based price or decide later.
- Generate listing text before pricing and a complete package after choosing a price.
- Review every fact and copy the result for manual publication on Tradera, Blocket or Vinted.

## Data and safety

- Schema 4 adds project names, explicit price decisions, archives and recoverable trash.
- Backup format 3 includes live and trashed projects while excluding secrets.
- Schema 3 migration is deterministic and retains verified rollback data.
- Seller-entered prices never receive invented confidence, ranges or AI/market claims.
- Numeric recommendations require two explicitly approved realized comparables.
- Follow-up advice is available for every pending recorded publication without calendar
  thresholds.

## Verification

- Full validation, browser acceptance and dependency audit.
- Linux AppImage visible-render smoke and v2.0.1 profile upgrade.
- Windows NSIS/portable package smoke and v2.0.1 installer upgrade.
- SHA-256 checksums and CycloneDX JSON SBOM.
- No open P0/P1, high-severity security or Dependabot findings at release preparation.

Release decisions use current evidence. Candidate age, waiting periods and elapsed time are not
release gates.

Windows packages remain unsigned. Verify downloads with `SHA256SUMS.txt`.
