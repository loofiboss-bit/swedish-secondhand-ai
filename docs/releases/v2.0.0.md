# Swedish Secondhand AI v2.0.0

V2 turns the app into a local-first Smart Seller Coach for Swedish private sellers.

## Highlights

- Multiple isolated item projects with schema 3 migration, image assets, validated full/compact
  backups, and read-only recovery instead of partial activation after migration failure.
- Deterministic next-best actions, category-aware fact gaps, and local photo quality guidance that
  never edits the original image.
- Editable exact/broad market searches, explicit asking/realized/unknown price provenance, and
  reproducible fast, balanced, and max-value price scenarios.
- Field-owned Tradera, Blocket, and Vinted listing drafts that preserve user edits, plus a
  transparent sell plan and structured manual-copy package.
- Local publication and outcome tracking with on-demand advice and category-segmented learning
  only after at least five verified own outcomes.

## Upgrade and data safety

Back up before upgrading. The v1 active draft and history are migrated idempotently into separate
projects. Schema 2 source keys remain untouched throughout the 2.0 series as a rollback source.
If migration, quota, or validation fails, the app opens read-only recovery with export instead of
activating a partial schema 3 dataset. Secrets are excluded from all backups.

HEIC/HEIF is not supported in v2.0.0. Use JPEG, PNG, or WebP. Publishing remains manual; the app
does not sign in to, automate, scrape, or publish directly to any marketplace.

## Downloads

- Linux x64 AppImage.
- Windows x64 NSIS installer and portable executable.
- CycloneDX JSON SBOM and `SHA256SUMS.txt` for downloadable artifacts.

Windows packages are intentionally unsigned because this project has no code-signing certificate.
Verify the matching SHA-256 checksum before use.

The maintainer explicitly removed beta and waiting-period prerequisites for this release. Stable
publication still requires green migration, security, Windows/Linux packaging and upgrade, and
artifact readback gates with no open P0/P1 or high-severity security finding.
