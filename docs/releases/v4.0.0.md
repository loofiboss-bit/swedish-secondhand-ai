# Swedish Secondhand AI v4.0.0

This stable release delivers the **Focused Seller Workspace** for preparing truthful,
copy-ready Tradera, Blocket, and Vinted listings from a local Windows or Linux desktop.

## A consistent seller path

- One derived readiness model powers the seller coach, project cards, workspace tabs, blocker
  counts, next actions, and ready-copy eligibility.
- A seller-entered price completes Price without making optional comparable research blocking.
- Evidence-based pricing still requires at least two user-approved realized comparables.
- Item focuses on required facts first, and Listing opens with the recommended marketplace
  preview and final copy action.

## Safer input and project recovery

- Quick start and Item share the same six-image, format, size, decode, local-assessment,
  rejection, removal, and stable-index handling.
- User-authored facts, seller locks, and edited listing fields remain authoritative.
- Rename, archive, trash, restore, and permanent deletion use accessible in-app dialogs.
- Serialized autosave and latest-request-wins project switching protect the active draft during
  rapid edits and persistence failures.

## Compatibility and boundaries

- Compatibility coverage preserves v2.0.1 and v3 facts, images, price decisions, listing edits,
  archive/trash state, and outcomes.
- Offline, Ollama, Gemini, and official Tradera data remain the supported analysis paths.
- Projects remain local unless the user explicitly invokes Gemini or Tradera.
- The app prepares listing copy but never signs in, scrapes, or publishes automatically.
- Repository documentation is English-only. The app retains Swedish and English localization,
  and supported marketplace listing output remains Swedish.

## Release verification

The release uses exact-source full validation, Chromium E2E, dependency audit, Windows and Linux
package and visible-launch smoke tests, stable-v3 upgrade tests, SHA-256 checksums, CycloneDX 1.6
SBOM, current security gates, and independent public artifact readback.

Candidate age, waiting periods, calendar thresholds, and elapsed time are not release gates.
Windows packages remain unsigned; verify downloads with `SHA256SUMS.txt`.
