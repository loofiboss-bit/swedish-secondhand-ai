# Swedish Secondhand AI wiki

Swedish Secondhand AI helps private sellers prepare Tradera, Blocket, and Vinted listings on a
Windows or Linux desktop. You can start offline without an account or API key. The app never
publishes on your behalf.

The current public build is **v4.0.0**. Back up existing projects before upgrading.

## Create a listing offline

1. Select **Start offline**.
2. Create a project with a name and item description.
3. Select **Identify item** and correct the facts.
4. Open **Price**, enter **Your price**, and select **Use my price**.
5. Open **Listing** and select **Update untouched fields**.
6. Review the marketplace tab and copy the structured package.

Publish the result manually. Verify condition, defects, missing parts, test status, and
authenticity before pasting the text into a marketplace.

## Price and privacy rules

You make the final price decision. Your own price is labelled clearly and receives no confidence
or market claim. An evidence-based recommendation needs at least two realized prices that you
approve. Asking prices remain context only.

Projects stay local unless you explicitly use Gemini or Tradera. Backups and safe diagnostics
exclude provider secrets. Read the repository's
[privacy policy](https://github.com/loofiboss-bit/swedish-secondhand-ai/blob/main/PRIVACY.md) for
the exact data flow.

## Get help

For a non-security bug, open a
[GitHub issue](https://github.com/loofiboss-bit/swedish-secondhand-ai/issues/new/choose) with the
app version, operating system, sanitized steps, and a reviewed safe diagnostic file. Never post
API keys, backup files, private URLs, images, or unpublished listing text.

Report vulnerabilities through
[GitHub private vulnerability reporting](https://github.com/loofiboss-bit/swedish-secondhand-ai/security/advisories/new).

Repository guides are versioned with the app and remain the source of truth when this page and a
release differ.
