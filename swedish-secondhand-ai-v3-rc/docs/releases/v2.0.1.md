# Swedish Secondhand AI v2.0.1

This maintenance release fixes Linux AppImage startup without changing user data or the v2
product boundary.

## Fixes

- Renderer assets and lazy chunks now use paths that work from Electron's packaged `file://`
  entry point.
- Release validation rejects root-relative packaged renderer assets.
- Linux package validation now opens the real AppImage renderer under Xvfb and requires visible
  application content instead of accepting process liveness alone.

## Upgrade and data safety

v2.0.1 uses the same schema 3 project data as v2.0.0. Existing projects, images, settings,
backups, protected secrets, and migration rollback records remain compatible. Backups continue to
exclude API keys.

## Downloads

- Linux x64 AppImage.
- Windows x64 NSIS installer and portable executable.
- CycloneDX JSON SBOM and `SHA256SUMS.txt` for downloadable artifacts.

Windows packages remain unsigned. Verify the matching SHA-256 checksum before use.
