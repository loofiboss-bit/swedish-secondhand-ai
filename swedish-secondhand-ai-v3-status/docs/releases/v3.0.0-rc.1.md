# Swedish Secondhand AI v3.0.0-rc.1

This release candidate introduces **Guided Selling**: a new seller can create a copy-ready local
listing with their own price without an account, API key or market data.

## Guided offline path

- Start offline immediately or create a clearly marked local example.
- Work through **Item → Price → Listing → Done**, with one primary next action.
- Choose **Your price**, optional evidence-based pricing, or decide later.
- Generate listing text before pricing and a complete package after choosing a price.
- Open the recommended marketplace first and switch with Tradera, Blocket and Vinted tabs.

## Data and safety

- Schema 4 adds explicit project names, price decisions, archives and recoverable trash.
- Backup format 3 includes live and trashed projects while excluding secrets.
- Schema 3 migration is deterministic and retains verified rollback data.
- Seller-entered prices never display confidence, ranges or AI/market claims.
- Numeric recommendations still require two explicitly approved realized comparables.
- Diagnostic export includes only versions, statuses and normalized error codes—never text,
  images, URLs or secrets.

## RC verification

- Unit/component/integration suite, E2E and automated WCAG 2.2 AA checks.
- Linux AppImage visible-render smoke test and v2.0.1 profile upgrade.
- Windows NSIS/portable package smoke test and v2.0.1 installer upgrade.
- SHA-256 checksums and CycloneDX JSON SBOM.

This prerelease starts the required seven-day RC observation period. Stable v3.0.0 will not be
published before the soak, clean-install/upgrade checks and open-finding gates are satisfied.

Windows packages remain unsigned. Verify downloads with `SHA256SUMS.txt`.
